HTML

  • HTML gives the webpage body and structure
  • The head element contains infoormation about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is the programming language primarily responsible for styling webpages
  • the link element is used to apply the css stylesheet to the html document
  • href is used to insert the specific path that your sheet or URL located in
  • A margin indicates how much space we want around the outside of an elemtent
  • A padding indicates how much space we want around the content inside an element
  • Elements displayed on a webpage are functionally a "box" due to this margin and padding is affected as such

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it automatically
  • git checkout main: returns you to the main/origin branch of the repository
  • git pull: pulls the files from the branch, and is also used to see if you are up to date with your local files
  • for the Terminal: cd .. returns you back one level in the folder directory
  • for the Terminal: touch "file-name" creates a new file in your current directory

JavaScript

  • A variable is a named container that allows us to store data in our code
  • JavaScript makes webpages interactive for the user
  • Control flow is the order in which a computer executes code in a script
  • Code is important in JavaScript as you should order functions and variables in the order that they should be executed
  • Every line of code ends with a semicolon
  • Console.log is used to log the variable to the web console on Chrome
  • The script element is used in HTML to link the JS file
