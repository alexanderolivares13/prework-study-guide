Prework Study Guide

HTML

  • HTML gives the webpage body and structure
  • The head element contains infoormation about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is the programming language primarily responsible for styling webpages

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it automatically
  • git checkout main: returns you to the main/origin branch of the repository
  • git pull: pulls the files from the branch, and is also used to see if you are up to date with your local files

JavaScript

  • JavaScript makes webpages interactive for the user
